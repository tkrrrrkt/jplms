# Product Overview

Japan LMS MVP — 日本語教師育成プラットフォーム。

在日中国人を主要ターゲットとし、日本語教師資格取得に必要な「420時間学習」をオンラインで完結できるLMS。中国での英語教師育成LMS（tesolchina.cn）の成功モデルをベースに、日本の資格制度に適合させたMVP。

**Architecture:** Single-tenant（1社専用）。マルチテナント不要。
**Client:** 日本イングレサ株式会社（代表取締役: 劉 志友）

## Core Capabilities

1. **420時間学習計測** — Heartbeat方式による厳密な時間記録。Page Visibility API + Vimeo SDK + BroadcastChannel APIの三重不正防止。
2. **Stripe決済 → 即時受講** — MVP: paygw_stripe + enrol_fee（標準プラグイン）。Phase 2: Custom Webhookで事前登録不要の即時受講。
3. **順次学習** — 単元ごとのロック機能。単元テスト70%合格で次単元アンロック。
4. **評価・添削** — 小テスト（自動採点）+ 課題提出（教案doc + 試講動画mp4）の講師10項目ルーブリック添削。
5. **成績管理** — 課題(60点) + 試験(40点) = 合計100点。合格60点以上、各項目60%以上。
6. **修了証自動発行** — 条件達成時のPDF証明書（検証番号付き）。

## Target Use Cases

- **学習者:** 動画視聴 → 小テスト → 課題提出 → 講師フィードバック → 修了証取得
- **講師:** 提出課題のオンライン添削・採点、学習進捗の確認
- **管理者:** 学習時間ログの監査、コース・受講者管理、売上管理

## Value Proposition

- **母国語サポート:** 中国語UI + 日本語学習コンテンツで学習障壁を最小化。
- **資格要件のデジタル証明:** 420時間の学習ログが監査に耐えうる精度で記録される。
- **低コストMVP:** Moodle活用により教育機能の9割を標準機能でカバー。初期予算120-150万円レンジ。
- **運営自動化:** 決済・登録・進捗管理の自動化で運営人件費を最小化。

## Scope Boundaries

**MVP (Phase 1):** 会員登録・決済（paygw_stripe）、動画学習、テスト・課題、420時間計測（local_timetrack）、修了証発行。

**Phase 2以降:** Custom Webhook決済（事前登録不要UX）、求人マッチング掲示板、登録日本語教員API連携、AI自動採点、証書オンライン検証。

## Legal & Compliance Requirements

### 特定商取引法（必須 — MVP）
- 専用の固定ページを作成し**全ページフッターからリンク**（Stripe審査の必須条件）
- ゲストユーザーでも閲覧可能に設定
- 必須表示: 販売業者名、所在地、電話番号、代表者、販売価格(税込)、支払方法、返品特約
- オンライン講座で2ヶ月超かつ5万円超の場合「特定継続的役務提供」に該当 → 8日間クーリングオフ

### インボイス制度（Phase 2 — B2B対応時）
- Stripe Dashboardで適格請求書発行事業者番号（T+13桁）を設定
- Stripe Invoicing機能で適格請求書の自動発行が可能
- 2029年9月までの経過措置期間あり

### 3Dセキュア
- Stripe Checkoutが自動処理（追加開発不要）
- 経産省義務化のEMV 3-D Secureに対応

### 420時間要件の規制的根拠
- 文化庁「日本語教員養成において必要とされる教育内容」— 全50項目の必須教育内容
- **1単位時間 = 最低45分**
- 2024年4月〜「登録日本語教員」国家資格化 → 時間管理の厳格化が進行中
- こども家庭庁見解: 「動画の流し見の純粋な技術的防止は極めて困難」→ **定期的な理解度チェック併用が必須**

## Success Metrics

- 420時間の計測ログが監査に耐えうる精度であること。
- 初期構築予算内で収まること。
- 特定商取引法・インボイス制度への法的準拠。
