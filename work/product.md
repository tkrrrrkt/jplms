# Product Context (Japan LMS MVP)

## 1. Vision & Background

**プロジェクト:** Japan LMS MVP（日本語教師育成プラットフォーム）

**背景:**

日本国内での外国人労働者増加に伴い、日本語教師が不足している。

中国で成功している英語教師育成LMS（tesolchina.cn）のノウハウを活かし、日本版の「日本語教師育成プラットフォーム」を構築する。

**ゴール:**

本格的なSaaS開発の前に、教育コンテンツ提供と学習管理（特に420時間要件）に特化したMVPを、短期間・低予算（150-200万円レンジ）で立ち上げる。

## 2. Core Value Proposition

- **Target User:** 日本語教師を目指す在日中国人、および日本語学習者。
    
- **Key Experience:**
    
    - オンライン完結での資格取得プロセス（学習→テスト→実技提出→認定）。
        
    - 母国語（中国語）サポートによる学習のしやすさ。
        
- **Business Value:**
    
    - 自動化された進捗管理と決済による運営コストの最小化。
        
    - 「420時間学習」という資格要件の厳密なデジタル証明。
        

## 3. Scope (MVP Phase)

**In Scope (必須機能):**

1. **会員登録・決済:** Stripe連携によるコース購入と即時受講開始。
    
2. **動画学習:** Vimeo埋め込み、単元ごとのロック機能（順次学習）。
    
3. **進捗管理 (Critical):** 420時間学習要件を満たすための厳密な滞在時間計測・ログ保存。
    
4. **評価:** 小テスト（自動採点）、課題提出（動画・教案）、講師による添削。
    
5. **修了証:** 条件達成時のPDF自動発行。
    

**Out of Scope (フェーズ2以降):**

- 求人マッチング掲示板。
    
- 登録日本語教員（国家資格）制度とのAPI直接連携（現時点ではCSVエクスポート対応等で凌ぐ）。
    
- AIによる課題の自動採点（フェーズ3）。
    

## 4. Success Metrics

- **機能要件:** 中国版（tesolchina.cn）の主要な学習フローが再現されていること。
    
- **非機能要件:** 420時間の計測ログが監査に耐えうる精度であること。
    
- **コスト:** 初期構築予算（サーバー・テーマ費含む）内で収めること。
